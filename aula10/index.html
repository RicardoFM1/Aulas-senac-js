Quer saber como fiz esse sistema todo de login e cadastro?

O simples objetivo desse site é de ter uma página inicial como essa, outra de login e a última para se cadastrar.

Todas são interligadas entre si e interagem entre si, possibilitanto fazer sistemas de cadastro ou login, que, exibe erros de acordocom o que está acontecendo ou toasts de sucesso caso tudo esteja certo e pronto para se cadastrar ou logar no site.

O legal de conseguir fazer isso tudo, que com criatividade e tempo, podemos fazer o que quiser com esse site. Podemos aprimorar seu CSS, deixando tudo mais intuitivo, bonito e chamativo, ou mais afundo ainda, usando ferramentas que auxiliam bastante na montagem e criação de sites mais complexos.

Ai você se pergunta, como você fez tudo isso? Para fazer isso tudo necessita de organização, lógica scripts, óbvio. Então, veja a seguir algumas das coisas que me possibilitaram fazer esse incrivel site:

  • Primeiro você precisa colocar uma meta de como será o site, como será seu layout, seus botões e
    principalmente, como irá funcionar.

  • Depois de determinar sua meta, você irá partir para a prática, utilizando CSS e HTML combinados
    para dar um toque de beleza para seu site.

  • E então por ultimo você pode partir para o os scripts que possibilitarão fazer tudo funcionar
    do jeito que é para funcionar. Essa parte pode ficar um pouco mais difícil, mas com calma e
    sabedoria chegamos longe.
